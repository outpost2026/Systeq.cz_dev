Produktové moduly
PRIMÁRNÍ PRODUKT
vcut-parser
První open-source parser pro binární formát .VCF (Ruida CNC controller). Kompletní reverzní inženýrství do 74B segmentové struktury. Extrahuje strojový čas, waste faktor a velocity profily přímo z kompilovaného souboru — bez nutnosti přístupu k CAM softwaru.
cut time waste factor velocity profiling Ruida RE
github.com/outpost2026/vcut-parser
DXF PIPELINE
CNC_2_LLM
Automatizovaná pipeline pro převod DXF souborů do strukturovaných dat. Výstup připravený pro ML modely, ERP kalkulace a automatizované nacenění zakázek bez manuálního přepisu geometrie.
DXF parsing ML-ready nacenění ERP
github.com/outpost2026/CNC_2_LLM
CAD BRIDGE
cad2llm
SketchUp/DAE → sémantický JSON. Řeší geometrické nekonzistence 4×4 transformačních matic při exportu COLLADA souborů. Validováno na 30 uzlech s 0% rate geometrických chyb v downstream analýzách.
SketchUp COLLADA spatial JSON matrix RE
github.com/outpost2026/cad2llm

API INTEGRACE

SYSTEQ engine běží jako autonomní cloudová platforma na Google Cloud Run. Klientský systém (Odoo, vlastní ERP, Python pipeline) volá zabezpečené REST endpointy a přijímá strukturovaný JSON — bez nutnosti instalace lokálního softwaru nebo přístupu ke CAM workstation.

Pro technickou specifikaci payloadů, příklady request/response cyklů a autentizační schéma nás kontaktujte přímo.
